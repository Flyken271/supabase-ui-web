## Installing

Install the component

```cli
npm install @supabase/ui
```

## Getting started

Import the component

```js
@import { Typography } from '@supabase/ui'
```

This component contains sub components which might be easier to use in variables.
Alternative import method can be to do the following.

```js
@import { Typography } from '@supabase/ui'

const { Title, Text, Link } from Typography

<Title level={3}>I am some text</Title>
<Text>I am some text</Text>
<Link>I am some link</Link>
```
